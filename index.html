Installation

<script type="module" src="https://unpkg.com/@annoyingmouse/wc-progress/wc-progress.js"></script>

Alternatively:

<script type="module" src="https://cdn.skypack.dev/@annoyingmouse/wc-progress/wc-progress.js"></script>

You can use it minified by adding using this instead:

<script type="module" src="[WHATEVER CDN]/@annoyingmouse/wc-progress/dist/wc-progress.min.js"></script>

Please remember to change [WHATEVER CDN] to your favourite CDN
